#  DecodeBot Pro

A professional Rule-Based AI Chatbot built using Python. DecodeBot Pro simulates natural conversations using predefined intents and responses while providing useful utilities like a calculator, chat history, date & time, and a colorful terminal interface.

---

##  Features

*  Rule-Based Conversational Chatbot
*  Colorful Terminal UI (Colorama)
*  Built-in Calculator
*  Current Date & Time
*  Chat History
*  Help Menu
*  About & Version Information
*  Fast and Lightweight
*  Command-Line Interface

---

##  Technologies Used

* Python 3
* Colorama

##  Installation

Clone the repository:

```bash
git clone https://github.com/Poojasaurus/Rule-base-Chatbot.git
```

Go to the project folder:

```bash
cd Rule-base-Chatbot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the chatbot:

```bash
python chatbot.py
```
